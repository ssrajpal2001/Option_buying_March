OI Exit Parameters
How OI Exit Works

OI is collected live from Upstox websocket every tick for the ATM ± strikes_range contracts.

Every check_interval_seconds, the monitor compares current OI to the OI at position entry:

Call OI rise > threshold → Exit Sell PE + Buy CE

Put OI rise > threshold → Exit Sell CE + Buy PE

OI data is collected unconditionally — toggling OI Exit Enabled only controls whether the exit action fires.
